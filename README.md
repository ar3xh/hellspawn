# Open Source Game? Meh
Real men test in production.

hellspawn (?) is a 2D survival game, built in GoDot from scratch.

## Contributing
 
The game is open to contributions. There's a lot left to do and not a lot of hands working on it.
 
If you want to help, some areas that could use work:
 
- enemy behavior and pathfinding
- hit detection and collision logic
- upgrade system design and implementation
- game feel stuff (screenshake, hitstop, knockback)
- balancing wave difficulty
No strict contribution guidelines yet. Just open a PR and we'll figure it out.
